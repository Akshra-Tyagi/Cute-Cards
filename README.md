# Cute Card Design

This project is a pair of cute and attractive cards designed using HTML and CSS.  
It demonstrates modern CSS styling techniques such as Flexbox layouts, hover effects, and capsule buttons.

# Features
- Responsive card layout using Flexbox
- Attractive card design with images
- Hover effects for interactive UI
- Capsule style buttons
- Clean and simple HTML structure
- Custom CSS styling

# Technologies Used
 HTML5
 CSS (Flexbox, Hover effects, Styling)

# Project Structure
index.html – main HTML file  
style.css – styling for the cards  
images – images used in the cards

# How to View
1. Download or clone the repository.
2. Open index.html in any web browser.
